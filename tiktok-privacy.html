Privacy Policy

become. Studio — last updated 2026

become. Studio is a personal desktop application. It is built for one user (its owner) and does not offer accounts to other people.

Data we do NOT collect

The application does not collect, store, sell, or share personal data from third parties. There is no analytics, no tracking, and no user database.

Local data

Access tokens for connected services (e.g. TikTok, GitHub) are stored locally on the owner's own computer and are never transmitted to anyone except the respective official API.

Images

Images prepared for a post are temporarily hosted on the owner's own GitHub Pages space so TikTok can fetch them, and are removed again after publishing.

Contact

This is a personal project. For questions, contact the account owner.
